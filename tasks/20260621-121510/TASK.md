# Add 'close <task-id1>[, <task-id2>]' to automatically close a or multiple tasks

- STATUS: CLOSED
- PRIORITY: 40
- TAGS: feature,cmdline-options

the title says it. automatically close a task by its id. can do that for multiple tasks at once such:

```bash
./tatr close <task-id1>,<task-id2>,...
```

or just one

```bash
./tatr close <task-id>
```
